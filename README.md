# wireguard 快捷配置

```bash
usage: gen.py [-h] [-i INDEX] [-n NAME] [-e ENDPOINT] [-p PORT] [--net NET]

options:
  -h, --help            show this help message and exit
  -i INDEX, --index INDEX
  -n NAME, --name NAME
  -e ENDPOINT, --endpoint ENDPOINT
                        服务端地址
  -p PORT, --port PORT  服务端口
  --net NET             服务的可以访问公网的网卡
```
